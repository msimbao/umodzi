# umodzi
An app and platform to provide fully offline educational content and quizzes to help high school students do well in Zambia

# To Run

1. Download Expo Go on your mobile device: [Expo Go](https://expo.dev/go)

2. Install Node Js if you don't already have it: [Node Js](https://www.google.com/url?sa=t&source=web&rct=j&opi=89978449&url=https://nodejs.org/en&ved=2ahUKEwjpxJXYxOeNAxXrJNAFHW19FwIQFnoECCgQAQ&usg=AOvVaw1dFXYHr5kNGMvANfSjX4lC)

3. move into directory via command line and enter the inner umodzi directory i.e (cd place-where-downloaded/umodzi/umodzi)

4. NOTE: You have to be in the INNER umodzi directory not the outer

5. run: npx expo start
  
6. Open the Expo Go app on phone
   
7. Scan QR Code in terminal
